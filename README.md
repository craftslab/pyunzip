*pyunzip* is a streaming unzip utility written in Python.



## Requirement

- Python >= 3.8



## Installation

### Ubuntu

```bash
apt update
apt install -y python3-dev python3-pip python3-setuptools
pip install pyunzip
```



### Windows

```
# Install Python, Microsoft Visual C++ and Windows SDK
pip install pyunzip
```



## Updating

```bash
pip install pyunzip --upgrade
```
